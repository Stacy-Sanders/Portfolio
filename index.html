Tech Training Tracker

Red Badge, our final project, we were to create a .NET Framework MVC 5 Web Application using n-tier architecture. I wanted to create a way to track my training courses, certifications, and the skills I have gained. This application allows a user to create their user information, the companies that offer the training courses and/or certifications, the training courses, the certifications, and their skills. This application could also be used by someone in a recruiter or management role as they could create multiple user profiles and the relevant information. The project was deployed through Azure App Services.

PetAdopterAPI

Blue Badge project - we were instructed to build a .NET Framework API Web Application using n-tier architecture. My three teammates and I decided to create a Pet Adopter API. We worked together to create data for domestic and exotic pets, adopters, and shelters. The application allows users to enter shelters, domestic pets, exotic pets, and adopters. With the API, a user is able to sort the pets by various properties including (but not limited to) if they are house trained, declawed, and/or sterile, the breed, and the shelter at which they are located. I was responsible for the domestic code and testing our endpoints with Postman.
